## 🚀 Technical Features

* **Precision Validation:** Custom JavaScript logic to validate email formats, providing immediate visual feedback and preventing empty submissions.
* **Mobile-First Layout:** A fluid, single-column structure designed to look perfect on devices of all sizes.
* **Visual Hierarchy:** Careful use of white space, typography, and iconography to guide the user's eye toward the call-to-action.
* **Social Integration:** A dedicated social media section featuring interactive hover states for community engagement.
* **Semantic HTML5:** Built with accessibility in mind, using proper form labels and structural tags for screen readers.

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic & Accessible)
- **Styling:** CSS3 (Flexbox, Custom Properties)
- **Logic:** Vanilla JavaScript (ES6+)
- **Deployment:** Vercel

## 🧠 Key Learnings

Building this landing page was an exercise in **UI simplicity** and **functional reliability**.

**Main Takeaways:**
- Implementing **regular expressions (RegEx)** for robust email validation.
- Mastering the use of **CSS Flexbox** for center-aligned, single-column layouts.
- Managing vertical rhythm and spacing to maintain a professional "Apple-style" minimalist aesthetic.
- Developing reusable CSS patterns for buttons and input states.

## 💻 Getting Started

    ```bash

    ```
2.  **Navigate to the folder:**
    ```bash
    cd ping-coming-soon-page
    ```

3.  **Run the project:**
    Open `index.html` in your browser.
